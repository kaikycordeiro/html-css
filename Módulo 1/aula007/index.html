Principais formatações

Negrito/Destaque

Nesta frase, temos um termo em negrito utilizando a tag <b> (não semântica).

Nesta frase, temos um termo em destaque usando a tag <strong> (semântico).

Itálico/Ênfase

Nesta frase tem os um termo em itálico usando a tag <i> (não semântica).

Nesta frase tem os um termo em itálico usando a tag <em> (semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele dever ser lido, mas não considerado. Tag usada: <del>

Texto Inserido

Podemos marcar um como texto inserido para dar uma ênfase e indicar que ele foi adicionado depois. Tag usada: <ins> (semântica).

Temos também o sublinhado usando a tag <u> (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x2+3 usando a tag <sup>.

Texto subscrito

Para inserir coisas do tipo H20 usando a tag <sub>.
